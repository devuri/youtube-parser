Plugin get a video screensnotes from youtube

== Description == 

Take away video screenshots from a link. Locate shortcode [previewparser] on the page.

== Installation ==

1. Upload `youtubeparser` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[previewparser]` shortcode in content
4. Set attribute 'width' to change width of plugin's header: 0 to 980px; full to 100%; other to set value in pixel
5. Set attribute 'uridef' to change default video
